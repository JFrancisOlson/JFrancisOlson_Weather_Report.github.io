API methods were used to capture data for this analysis from Openweathermap.org. The results were evaluated by temperature, humidity, cloudiness, and windspeed charted against latitudes. The data was collected from 606 cities worldwide on 10/9/2018 using Citipy to randomly generate the sample of cities based on latitude and longitude for this data set. A list compression loop was used to collect the data with a 1 second per loop throttle to prevent oversampling issues with the vendor. The data was parsed using json methods and cast into lists and then assembled into a dataframe (see figure 1). Plots were constructed from the dataframe sample to analyze results. Observations based on this series of scatter plots to showcase the following relationships. Units are imperial in this dataset (Fahrenheit, humidity %, windspeed in mph ,% cloudiness).
